🤖 Algoritmos, IA e Dados

Algoritmo

O que é: Conjunto bem definido de instruções para resolver um problema.

Exemplo: O YouTube determina recomendações seguindo um algoritmo.

Big Data

O que é: Grandes volumes de dados em alta velocidade e variedade.

Exemplo: Empresas de e-commerce analisam petabytes de histórico de navegação.

Data Mining

O que é: Extração de padrões de grandes bases de dados.

Exemplo: Redes de supermercado usam data mining para identificar padrões de compra.

Data Lake

O que é: Repositório de dados brutos.

Exemplo: Empresas de IoT armazenam dados de sensores para análises futuras.

IA (Inteligência Artificial)

O que é: Campo que desenvolve sistemas com capacidade de aprendizado e tomada de decisão.

Exemplo: Assistentes como Alexa e ChatGPT.

Machine Learning

O que é: Subcampo da IA onde algoritmos aprendem padrões a partir de dados.

Exemplo: Netflix sugere filmes baseado no histórico.

IoT Analytics

O que é: Análise de dados gerados por dispositivos IoT.

Exemplo: Empresas de energia monitoram consumo em tempo real.

Análise Heurística

O que é: Identificação de ameaças baseada em regras aproximadas.

Exemplo: Antivírus detectam vírus desconhecidos por comportamento.

Metadados

O que é: Dados que descrevem outros dados.

Exemplo: Fotos digitais contêm metadados de GPS e data.

Aprendizado Supervisionado

O que é: ML com dados rotulados.

Exemplo: Classificação de e-mails como spam ou não.

Aprendizado Não Supervisionado

O que é: ML sem rótulos, busca padrões.

Exemplo: Agrupamento de clientes por comportamento.

Data Warehouse

O que é: Armazém de dados estruturados para BI.

Exemplo: Dados de vendas e marketing consolidados para dashboards.

Data Visualization

O que é: Representação visual de dados.

Exemplo: Gráficos no Tableau ou Power BI.

Redes Neurais

O que é: Modelo inspirado no cérebro, com neurônios artificiais.

Exemplo: Reconhecimento facial.

Aprendizado por Reforço

O que é: Agente aprende por recompensas.

Exemplo: AlphaGo.

GPT

O que é: Modelo de linguagem transformer.

Exemplo: ChatGPT.

Redes Adversárias (GANs)

Exemplo: Geração de imagens falsas realistas.

Processamento de Linguagem Natural (PLN)
