Dicionário Online de Tecnologia

Explore termos do universo da tecnologia, organizados por categorias ou listados completamente.

Redes

DNS (Sistema de Nomes de Domínio)

Sistema que traduz nomes de sites para endereços IP.

Ethernet

Tecnologia usada para conectar dispositivos a uma rede local via cabo.

Firewall

Sistema de segurança que monitora e controla o tráfego de rede.

Mesh Network

Rede onde cada dispositivo se conecta a outros, formando uma malha de comunicação.

VPN (Rede Privada Virtual)

Tecnologia que cria uma conexão segura e privada entre o dispositivo e a internet.

Programação

API (Interface de Programação de Aplicativos)

Conjunto de regras que permite a comunicação entre sistemas.

Compilador

Programa que traduz código fonte em linguagem de máquina.

Framework

Estrutura pré-definida para facilitar o desenvolvimento de aplicativos ou sistemas.

Git

Sistema de controle de versão para rastrear mudanças no código.

Singleton

Padrão de design que garante que uma classe tenha apenas uma instância.

Segurança

Criptografia

Método de proteger informações transformando-as em código.

Phishing

Tentativa de enganar usuários para roubar informações pessoais.

Token

Código usado para autenticação e segurança.

Zero-Day

Vulnerabilidade de software descoberta antes de ser corrigida.

Botnet

Rede de computadores infectados controlados por hackers.

Hardware

CPU (Unidade Central de Processamento)

O "cérebro" do computador que executa comandos e processa dados.

GPU (Unidade de Processamento Gráfico)

Componente especializado em processar gráficos e vídeos.

Memória RAM

Tipo de memória temporária usada para armazenar dados enquanto são processados.

Microchip

Componente eletrônico miniaturizado que executa funções computacionais.

SSD (Unidade de Estado Sólido)

Dispositivo de armazenamento mais rápido e confiável que o HDD.

Termos Completos

1. Hardware

O que é: Componentes físicos de um computador ou dispositivo eletrônico, como monitor, teclado, mouse, processador, etc.

Exemplo: O celular que você segura é um exemplo de hardware.

2. Software

O que é: Programas e sistemas que executam tarefas em dispositivos eletrônicos.

Exemplo: Aplicativos como WhatsApp ou Word são softwares.

3. Sistema Operacional (SO)

O que é: O software principal que gerencia o hardware e outros programas de um dispositivo.

Exemplo: Windows, macOS, Android, iOS.

4. Navegador

O que é: Programa usado para acessar a internet.

Exemplo: Google Chrome, Safari, Mozilla Firefox.

5. Wi-Fi

O que é: Tecnologia que permite a conexão à internet sem cabos.

Exemplo: Redes sem fio usadas em casa, no trabalho ou em locais públicos.

6. Nuvem (Cloud)

O que é: Armazenamento e acesso de dados pela internet em vez de usar um dispositivo físico.

Exemplo: Google Drive, iCloud.

7. Inteligência Artificial (IA)

O que é: Tecnologia que simula a inteligência humana em máquinas.

Exemplo: Assistentes virtuais como Alexa e ChatGPT.

8. Realidade Virtual (VR)

O que é: Ambiente simulado criado por computador, acessado com dispositivos como óculos VR.

Exemplo: Jogos imersivos ou tours virtuais.

9. Realidade Aumentada (AR)

O que é: Tecnologia que adiciona elementos digitais ao mundo real, normalmente através de câmeras.

Exemplo: Filtros do Instagram ou jogos como Pokémon GO.

10. Algoritmo

O que é: Conjunto de regras ou passos para resolver um problema ou realizar uma tarefa.

Exemplo: Como o YouTube escolhe vídeos para recomendar a você.

11. Rede Social

O que é: Plataformas digitais para interação e compartilhamento de conteúdo.

Exemplo: Facebook, Instagram, TikTok.

12. URL

O que é: Endereço de um site na internet.

Exemplo: https://www.google.com é uma URL.

13. Servidor

O que é: Computador ou sistema que fornece recursos, dados ou serviços a outros computadores.

Exemplo: O Netflix usa servidores para transmitir filmes.

14. Banco de Dados

O que é: Local onde informações são armazenadas e organizadas para fácil acesso.

Exemplo: Cadastros de clientes em um sistema de loja.

15. Criptomoeda

O que é: Moeda digital que usa criptografia para segurança.

Exemplo: Bitcoin, Ethereum.

16. Blockchain

O que é: Tecnologia usada para registrar transações digitais de forma segura e descentralizada.

Exemplo: Base de funcionamento das criptomoedas.

17. Código-Fonte

O que é: Conjunto de instruções escritas em linguagem de programação que define como um software funciona.

Exemplo: Os aplicativos são criados com código-fonte.

18. Inteligência Coletiva

O que é: Compartilhamento de conhecimento entre um grupo de pessoas usando tecnologia.

Exemplo: Wikipedia, fóruns online.

19. Segurança Cibernética

O que é: Práticas e ferramentas para proteger dispositivos e redes contra ataques e acessos não autorizados.

Exemplo: Uso de antivírus ou autenticação em dois fatores.

20. Internet das Coisas (IoT)

O que é: Conexão de dispositivos físicos à internet para coletar e compartilhar dados.

Exemplo: Geladeiras inteligentes ou lâmpadas controladas por aplicativos.

21. Download e Upload

Download: Transferir arquivos da internet para seu dispositivo.

Upload: Enviar arquivos do seu dispositivo para a internet.

Exemplo: Baixar uma música (download) e postar uma foto no Instagram (upload).

22. Spam

O que é: Mensagens indesejadas ou irrelevantes enviadas para muitas pessoas, geralmente por e-mail.

Exemplo: Propagandas excessivas no seu e-mail.

23. Malware

O que é: Software malicioso projetado para danificar ou acessar informações de um sistema.

Exemplo: Vírus de computador.

24. Bug

O que é: Erro ou falha em um software ou sistema.

Exemplo: Quando um aplicativo fecha sozinho ou não responde.

25. Inteligência de Dados (Big Data)

O que é: Análise de grandes volumes de dados para obter informações úteis.

Exemplo: Empresas usam Big Data para entender o comportamento dos clientes.

26. Processador (CPU)

O que é: O "cérebro" do computador que executa comandos e processa dados.

Exemplo: Intel Core i7, AMD Ryzen.

27. Memória RAM

O que é: Tipo de memória temporária usada para armazenar dados enquanto são processados.

Exemplo: Mais RAM significa que você pode abrir mais aplicativos sem lentidão.

28. Disco Rígido (HDD) e SSD

O que é: Dispositivos de armazenamento de dados. SSDs são mais rápidos e confiáveis que HDDs.

Exemplo: Armazenar fotos e vídeos no computador.

29. API (Interface de Programação de Aplicativos)

O que é: Conjunto de regras que permite que diferentes sistemas se comuniquem.

Exemplo: O Google Maps API permite integrar mapas em sites.

30. Firmware

O que é: Software embutido em hardware que controla seu funcionamento.

Exemplo: O firmware de uma impressora garante que ela funcione corretamente.

31. Open Source (Código Aberto)

O que é: Software cujo código-fonte é público e pode ser modificado por qualquer pessoa.

Exemplo: O sistema operacional Linux é open source.

32. Backend

O que é: Parte "invisível" de um sistema, onde a lógica e os dados são processados.

Exemplo: Os servidores que mantêm um site funcionando.

33. Frontend

O que é: Parte "visível" e interativa de um sistema que o usuário acessa.

Exemplo: Design e botões em uma página web.

34. Cache

O que é: Memória temporária que armazena dados usados com frequência para acesso mais rápido.

Exemplo: Seu navegador salva imagens de sites para carregar mais rápido na próxima visita.

35. Deep Web

O que é: Parte da internet não acessível por mecanismos de busca padrão.

Exemplo: Bancos de dados privados ou sites protegidos por login.

36. Dark Web

O que é: Subconjunto da Deep Web, acessado com softwares específicos como Tor, geralmente associado a atividades ilegais.

Exemplo: Não recomendado para usuários comuns.

37. Data Mining

O que é: Processo de analisar grandes volumes de dados para descobrir padrões ou tendências.

Exemplo: Empresas analisam compras de clientes para oferecer promoções personalizadas.

38. Machine Learning (Aprendizado de Máquina)

O que é: Subcampo da IA onde máquinas aprendem a realizar tarefas sem serem programadas diretamente.

Exemplo: Sistemas de recomendação como o Netflix.

39. Criptografia

O que é: Método de proteger informações transformando-as em código.

Exemplo: Transações bancárias online são criptografadas para segurança.

40. DNS (Sistema de Nomes de Domínio)

O que é: Sistema que traduz nomes de sites (ex.: google.com) para endereços IP.

Exemplo: Sem DNS, seria necessário digitar o IP 8.8.8.8 em vez de "google.com".

41. VPN (Rede Privada Virtual)

O que é: Tecnologia que cria uma conexão segura e privada entre seu dispositivo e a internet.

Exemplo: Usada para proteger dados em redes Wi-Fi públicas.

42. Pixel

O que é: Menor unidade de uma imagem digital.

Exemplo: Uma tela 1920x1080 tem 2.073.600 pixels.

43. DPI (Pontos por Polegada)

O que é: Medida de resolução usada em imagens e impressoras.

Exemplo: Fotos de alta qualidade geralmente têm 300 DPI.

44. Driver

O que é: Software que permite que o sistema operacional se comunique com hardware específico.

Exemplo: Driver de impressora para conectar ao computador.

45. IoT Gateway

O que é: Dispositivo que conecta sensores e dispositivos IoT à internet.

Exemplo: Um roteador IoT que controla luzes e termostatos inteligentes.

46. Token

O que é: Código usado para autenticação e segurança.

Exemplo: Tokens são enviados por SMS para login seguro.

47. Data Center

O que é: Local físico onde servidores armazenam e processam dados.

Exemplo: Empresas como Google e Amazon possuem enormes data centers.

48. 5G

O que é: Quinta geração de redes móveis, com maior velocidade e menor latência.

Exemplo: Permite streaming de vídeos em alta definição sem interrupções.

49. SaaS (Software como Serviço)

O que é: Modelo de software acessado pela internet, geralmente por assinatura.

Exemplo: Microsoft Office 365, Google Workspace.

50. Phishing

O que é: Tentativa de enganar usuários para roubar informações pessoais.

Exemplo: E-mails falsos que imitam bancos pedindo suas senhas.

51. Hotspot

O que é: Ponto de acesso Wi-Fi compartilhado, geralmente público.

Exemplo: Wi-Fi gratuito em aeroportos.

52. Cookies

O que é: Pequenos arquivos armazenados por sites no seu navegador para lembrar informações.

Exemplo: Um site que lembra sua preferência de idioma.

53. Renderização

O que é: Processo de gerar imagens ou vídeos a partir de modelos digitais.

Exemplo: Usado na criação de gráficos 3D em jogos e filmes.

54. HTTP e HTTPS

O que é: Protocolos de comunicação para acessar sites. O "S" no HTTPS significa que a conexão é segura.

Exemplo: Sites de banco usam HTTPS.

55. Ping

O que é: Teste usado para medir a latência (tempo de resposta) entre dois dispositivos na rede.

Exemplo: Jogos online usam "ping" para medir atrasos.

56. IoT Analytics

O que é: Análise de dados coletados por dispositivos IoT.

Exemplo: Sensores inteligentes monitorando consumo de energia.

57. Metadados

O que é: Informações sobre dados, como autor, data de criação e formato.

Exemplo: Uma foto digital inclui metadados como local e data.

58. Firewall

O que é: Sistema de segurança que monitora e controla o tráfego de rede, bloqueando acessos não autorizados.

Exemplo: Empresas usam firewalls para proteger suas redes internas.

59. Bit e Byte

Bit: Menor unidade de dados em informática, representando 0 ou 1.

Byte: Conjunto de 8 bits.

Exemplo: Um arquivo de texto pode ter 2 KB (2.000 bytes).

60. BIOS

O que é: Sistema básico que inicia o hardware do computador e carrega o sistema operacional.

Exemplo: Quando você liga o computador, o BIOS prepara o hardware para o SO.

61. SDK (Kit de Desenvolvimento de Software)

O que é: Conjunto de ferramentas e bibliotecas para criar aplicativos em uma plataforma específica.

Exemplo: O SDK do Android é usado para criar aplicativos móveis.

62. Latência

O que é: Tempo que um dado leva para viajar de um ponto a outro em uma rede.

Exemplo: Jogos online podem ter atrasos devido à alta latência.

63. Overclock

O que é: Ato de aumentar a velocidade de componentes do computador, além do recomendado.

Exemplo: Gamers usam overclock para melhorar o desempenho.

64. FTP (Protocolo de Transferência de Arquivos)

O que é: Protocolo usado para transferir arquivos entre computadores.

Exemplo: Utilizado para carregar sites em servidores.

65. Router (Roteador)

O que é: Dispositivo que distribui a conexão de internet para outros dispositivos.

Exemplo: O roteador Wi-Fi que você usa em casa.

66. Ethernet

O que é: Tecnologia usada para conectar dispositivos a uma rede local (LAN) via cabo.

Exemplo: Computadores de escritório usam cabos Ethernet.

67. DNS Spoofing

O que é: Ataque que redireciona usuários para sites falsos, alterando informações do DNS.

Exemplo: Hackers podem redirecionar você para um site falso que imita seu banco.

68. Virtualização

O que é: Criação de versões virtuais de dispositivos ou recursos, como servidores ou sistemas.

Exemplo: Rodar Windows em um Mac via máquina virtual.

69. Proxy

O que é: Servidor intermediário entre o usuário e a internet, usado para filtrar ou proteger conexões.

Exemplo: Escolas usam proxies para bloquear sites não autorizados.

70. QR Code

O que é: Código bidimensional escaneável para acessar informações ou links.

Exemplo: Usado para pagamentos por aproximação.

71. Kernel

O que é: Parte central de um sistema operacional que gerencia recursos do hardware.

Exemplo: O kernel do Linux é usado em muitos sistemas.

72. Load Balancer (Balanceador de Carga)

O que é: Dispositivo ou software que distribui tráfego entre vários servidores.

Exemplo: Grandes sites usam load balancers para lidar com milhões de acessos.

73. CLI (Interface de Linha de Comando)

O que é: Método de interagir com um computador usando comandos de texto.

Exemplo: Desenvolvedores usam o terminal Linux.

74. GPU (Unidade de Processamento Gráfico)

O que é: Componente especializado em processar gráficos e vídeos.

Exemplo: Uma NVIDIA GeForce para jogos.

75. IP Público e IP Privado

IP Público: Endereço usado na internet.

IP Privado: Endereço usado em redes locais.

Exemplo: Seu roteador atribui um IP privado ao seu celular.

76. Tokenização

O que é: Processo de dividir texto em partes menores (palavras/frases).

Exemplo: Usado em processamento de linguagem natural.

77. Microchip

O que é: Componente eletrônico miniaturizado com funções computacionais.

Exemplo: Chips em smartphones, cartões bancários, IoT.

78. SEO (Otimização para Motores de Busca)

O que é: Técnicas para melhorar a posição de sites em resultados de busca.

Exemplo: Usar palavras-chave para aparecer no Google.

79. RAM Virtual

O que é: Parte do armazenamento usada como RAM quando a RAM real está cheia.

Exemplo: Sistemas criam arquivos de troca (swap).

80. JSON (JavaScript Object Notation)

O que é: Formato leve para troca de dados entre sistemas.

Exemplo: APIs retornam dados em JSON.

81. WebSocket

O que é: Protocolo para comunicação bidirecional em tempo real entre navegador e servidor.

Exemplo: Usado em chats online.

82. Data Lake

O que é: Repositório que armazena dados em formato bruto e não estruturado.

Exemplo: Empresas guardam dados de sensores IoT.

83. Edge Computing

O que é: Processamento de dados próximo da fonte para reduzir latência.

Exemplo: Câmeras inteligentes processam dados localmente.

84. RAID

O que é: Método de combinar múltiplos discos para redundância/desempenho.

Exemplo: Usado em servidores para backups.

85. Worm (Vírus de Rede)

O que é: Malware que se replica automaticamente em redes.

Exemplo: Worms podem espalhar ransomware.

86. Deepfake

O que é: Vídeos ou áudios falsos criados por IA.

Exemplo: Vídeos simulando celebridades.

87. PaaS (Plataforma como Serviço)

O que é: Modelo de nuvem que oferece plataforma para desenvolvimento de aplicativos.

Exemplo: Heroku, Google App Engine.

88. CLI Bot

O que é: Ferramenta automatizada que executa tarefas via linha de comando.

Exemplo: Um bot CLI pode organizar arquivos.

89. Microcontrolador

O que é: Pequeno computador em um único chip que controla dispositivos.

Exemplo: Arduino e Raspberry Pi.

90. Compilador

O que é: Programa que traduz código fonte em linguagem de máquina.

Exemplo: O GCC compila código C.

91. Shell

O que é: Interface para interagir com o sistema operacional.

Exemplo: Bash no Linux.

92. Framework

O que é: Estrutura pré-definida para desenvolvimento de aplicativos/sistemas.

Exemplo: Django, React.

93. Modem

O que é: Dispositivo que conecta redes locais à internet.

Exemplo: Modem de banda larga.

94. Padrão USB

O que é: Tecnologia para conexão e transferência de dados entre dispositivos.

Exemplo: Pen drives, mouses e teclados.

95. Bitrate

O que é: Quantidade de dados por segundo em áudio ou vídeo.

Exemplo: Vídeos de alta qualidade têm maior bitrate.

96. Stack

O que é: Conjunto de tecnologias usadas para criar aplicativos.

Exemplo: MERN (MongoDB, Express, React, Node.js).

97. Latência de Disco

O que é: Tempo para acessar dados no disco.

Exemplo: SSDs têm menor latência que HDDs.

98. Endereço MAC

O que é: Identificador único de dispositivos de rede.

Exemplo: Wi-Fi do seu celular tem um MAC único.

99. Cluster

O que é: Conjunto de computadores que trabalham juntos como um só.

Exemplo: Usado em data centers para alto desempenho.

100. Phablet

O que é: Dispositivo que combina smartphone e tablet.

Exemplo: Galaxy Note.

101. Formatação

O que é: Processo de apagar e preparar um disco para uso.

Exemplo: Formatar um pen drive.

102. Wearables

O que é: Dispositivos eletrônicos vestíveis.

Exemplo: Smartwatches, pulseiras fitness.

103. Near Field Communication (NFC)

O que é: Tecnologia para troca de dados entre dispositivos próximos.

Exemplo: Pagamento por aproximação em celulares.

104. Emulação

O que é: Imitar o comportamento de um sistema em outro.

Exemplo: Emuladores de jogos retrô.

105. Overhead

O que é: Recursos adicionais necessários para executar uma tarefa.

Exemplo: Software com overhead alto consome mais memória.

106. Render Farm

O que é: Conjunto de computadores para renderizar gráficos complexos.

Exemplo: Usado em filmes de animação.

107. Deep Link

O que é: Link que leva diretamente a uma página ou seção específica.

Exemplo: Links que abrem diretamente um produto em um app.

108. Git

O que é: Sistema de controle de versão para rastrear mudanças no código.

Exemplo: GitHub usa Git.

109. LAMP

O que é: Conjunto de tecnologias: Linux, Apache, MySQL, PHP.

Exemplo: Amplamente usado em servidores web.

110. Multithreading

O que é: Execução de várias tarefas simultâneas em um processador.

Exemplo: Jogos utilizam multithreading para desempenho.

111. BIOS UEFI

O que é: Sucessor moderno do BIOS, mais rápido e com interface gráfica.

Exemplo: PCs atuais usam UEFI.

112. RAID 0/1/5

O que é: Configurações RAID com diferentes combinações de desempenho e redundância.

Exemplo: RAID 1 duplica dados.

113. Token NFT

O que é: Certificado digital único em blockchain.

Exemplo: NFTs são usados em arte digital.

114. Hotfix

O que é: Atualização rápida para corrigir um problema crítico.

Exemplo: Jogos lançam hotfixes para bugs.

115. IPv6

O que é: Nova versão do protocolo IP, com mais endereços que IPv4.

Exemplo: Dispositivos modernos migram para IPv6.

116. Load Time

O que é: Tempo para carregar uma página ou app.

Exemplo: Sites rápidos carregam em menos de 3 segundos.

117. Streaming

O que é: Transmissão de áudio/vídeo sem baixar.

Exemplo: Netflix, Spotify.

118. Botnet

O que é: Rede de computadores infectados controlados por hackers.

Exemplo: Usadas para ataques DDoS.

119. Análise Heurística

O que é: Método de segurança cibernética para detectar padrões suspeitos.

Exemplo: Antivírus usam análise heurística.

120. DLL (Dynamic-Link Library)

O que é: Arquivos com código/dados usados por múltiplos programas no Windows.

Exemplo: Arquivos .dll essenciais para aplicativos.

121. ASCII

O que é: Código padrão para representação de texto em computadores.

Exemplo: "A" tem um código ASCII.

122. Hypervisor

O que é: Software/hardware que cria e gerencia máquinas virtuais.

Exemplo: VMware, Hyper-V.

123. DNS Cache

O que é: Memória temporária de consultas DNS.

Exemplo: Sites carregam mais rápido após a primeira visita.

124. Bloatware

O que é: Software pré-instalado desnecessário ou pesado.

Exemplo: Apps pré-instalados em smartphones.

125. Dual-Boot

O que é: Ter dois sistemas operacionais no mesmo dispositivo.

Exemplo: Windows e Linux no mesmo PC.

126. Zero-Day

O que é: Vulnerabilidade descoberta antes da correção.

Exemplo: Hackers exploram falhas zero-day.

127. Mesh Network

O que é: Rede onde cada dispositivo conecta-se a outros, formando uma malha.

Exemplo: Roteadores mesh em casas grandes.

128. TLD (Top-Level Domain)

O que é: Extensão no final de um domínio.

Exemplo: .com, .org, .edu.

129. Virtual Private Cloud (VPC)

O que é: Ambiente isolado na nuvem.

Exemplo: Usado para segurança na nuvem.

130. Packet Loss

O que é: Perda de pacotes de dados na transmissão.

Exemplo: Pode causar falhas em chamadas de vídeo.

131. Load Testing

O que é: Testar desempenho do sistema sob carga pesada.

Exemplo: Testes antes da Black Friday.

132. CAPTCHA

O que é: Teste para diferenciar humanos de bots.

Exemplo: "Selecione todas as imagens com semáforos."

133. PoE (Power over Ethernet)

O que é: Alimentar dispositivos via cabo Ethernet.

Exemplo: Câmeras IP usam PoE.

134. Metadata

O que é: Dados sobre dados.

Exemplo: Fotos com localização e data.

135. SaaS (Software como Serviço)

O que é: Aplicações acessíveis pela internet, sem instalação local.

Exemplo: Google Docs, Salesforce.

136. Singleton

O que é: Padrão de design que limita uma classe a uma única instância.

Exemplo: Usado em gerenciadores de configuração.

137. Blockchain Explorer

O que é: Ferramenta para visualizar transações no blockchain.

Exemplo: Exploradores de Bitcoin.

138. NAT (Network Address Translation)

O que é: Permite que vários dispositivos usem um só IP público.

Exemplo: Roteadores domésticos usam NAT.

139. Payload

O que é: Dados transmitidos em um pacote de rede.

Exemplo: Em ataques, o payload pode ser malware.

140. Quantum Computing

O que é: Computação usando princípios da física quântica.

Exemplo: Google e IBM pesquisam computadores quânticos.

141. Cold Storage

O que é: Armazenamento offline para maior segurança.

Exemplo: Carteiras físicas de Bitcoin.

142. Thin Client

O que é: Dispositivo leve que depende de servidores.

Exemplo: Usado em call centers.

143. Codec

O que é: Codifica/decodifica áudio/vídeo.

Exemplo: H.264, MP3.

144. Breadcrumb

O que é: Navegação hierárquica que mostra a localização do usuário.

Exemplo: "Home > Produtos > Eletrônicos".

145. Garbage Collection

O que é: Liberação automática de memória não utilizada.

Exemplo: Java usa garbage collection.

146. Web Scraping

O que é: Extrair dados de sites.

Exemplo: Coletar preços de produtos online.

147. Heatmap

O que é: Visualização que mostra onde usuários clicam mais.

Exemplo: Análise de comportamento em sites.

148. ASIC (Application-Specific Integrated Circuit)

O que é: Hardware especializado para uma tarefa específica.

Exemplo: Usado em mineração de criptomoedas.

149. DPI Scaling

O que é: Ajuste de elementos gráficos para melhor legibilidade em alta resolução.
